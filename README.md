## 📚 Projects

### 1. Smart Library Management System
**Language:** C++  
**File:** `library_system.cpp`

**Description:**
A console-based system that manages books, users, and borrowing/returning operations efficiently using data structures.

**Problem Statement:**
Design a system for a library where users can add books, search books, issue books, and return books with proper tracking.

**Features:**
- ✅ Add Book
- ✅ Remove Book
- ✅ Search Book (by ID and Title)
- ✅ Issue Book
- ✅ Return Book
- ✅ Display all books
- ✅ Display issued books list
- ✅ Sort books by ID or Title

**Key Requirements:**
- Use classes and OOP principles
- Store books using vectors
- Implement search functionality
- Track issued books
- Handle edge cases (book not found, already issued, etc.)
- Add comments explaining logic

**Constraints:**
- Max 1000 books
- Unique book ID for each book

**How to Run:**
```bash
g++ library_system.cpp -o library
./library
```

**Technologies Used:**
- C++ (OOP)
- Vector data structure
- Console I/O
